# Data Safety & Privacy Nutrition Labels

This document outlines the data collected by AI Fitness Coach and how it is used, in a format suitable for app store privacy nutrition labels.

## Data Collection

| Data Type | Collected | Optional | Why is this data collected? |
|---|---|---|---|
| **Personal Info** | | | |
| Name | Yes | No | To personalize the user experience. |
| Email Address | Yes | No | For account creation and authentication. |
| **Health & Fitness** | | | |
| Health Info (height, weight, etc.) | Yes | No | To provide personalized fitness plans and calorie estimates. |
| Fitness Info (workouts, reps) | Yes | No | To track progress and provide real-time feedback. |
| **Photos & Videos** | | | |
| Videos | Yes | No | For real-time pose detection. Videos are processed on-device and not stored. |
| **Device & App Info** | | | |
| Device ID | Yes | Yes | For analytics and crash reporting. |
| App Performance Data | Yes | Yes | To monitor and improve app stability. |

## Data Sharing

We do **not** share any personal or health data with third parties, except for anonymized and aggregated data used for analytics and research.

## Data Retention

- **Account Data:** Retained as long as the account is active.
- **Workout Data:** Retained for historical progress tracking.
- **Videos:** Not stored or retained.

Users can request data deletion at any time through the app's privacy settings.
